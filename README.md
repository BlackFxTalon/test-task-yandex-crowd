# Клуб четырех коней

Адаптивный лендинг Международного васюкинского турнира по шахматам по мотивам тестового макета. Проект сверстан на чистых HTML, CSS и JavaScript без сборщика и внешних runtime-зависимостей.

## Что реализовано

- Первый экран с фоновым изображением, логотипом, заголовком и CTA-ссылками.
- Бегущие строки с новостными фразами.
- Информационные блоки о лекции и сеансе одновременной игры.
- Блок этапов преображения Васюков:
  - один источник текстового контента в `ol`;
  - CSS-нумерация через counter и псевдоэлемент;
  - десктопная grid-раскладка;
  - мобильная карусель через `transform: translateX`.
- Блок участников турнира:
  - карточки генерируются из данных в `script.js`;
  - бесконечная карусель с клонами;
  - анимация ленты через `transform: translateX`;
  - 3 карточки на десктопе, 2 на планшете, 1 на мобильных.
- Поддержка WebP для изображений с PNG fallback в HTML и JS.
- WebP-фоны в CSS.

## Запуск

Проект статический. Можно открыть `index.html` напрямую в браузере.

После запуска страница будет доступна по адресу:

```text
http://127.0.0.1:4173/index.html
```

## Технологии
